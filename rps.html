Rock Paper Scissors Game

The goal of the game is to choose a move (rock, paper, or scissors) that beats the computer's randomly chosen move based on the following rules:

View code

  1. Game Initialization :

    - The game starts and runs continuously in a loop, allowing you to play multiple rounds until you decide to stop.

  2. Choices :

    - There are three possible choices for both the player and the computer: rock, paper, and scissors.

  3. Computer's Move :

    - The computer randomly selects one of the three choices (rock, paper, or scissors) at the beginning of each round.

  4. Player's Move :

    - You, the player, are prompted to input your choice. The game ensures that your input is valid (i.e., it must be either rock, paper, or scissors).

  5. Displaying Moves :

    - Once both the computer and the player have made their choices, the game displays both choices.

  6. Determining the Winner :

    - The game compares the player's choice with the computer's choice to determine the winner:
    - If both choices are the same, it’s a tie.
    - If the player's choice is rock and the computer's choice is scissors, the player wins (rock crushes scissors). If the computer's choice is paper, the player loses (paper covers rock).
    - If the player's choice is paper and the computer's choice is rock, the player wins (paper covers rock). If the computer's choice is scissors, the player loses (scissors cut paper).
    - If the player's choice is scissors and the computer's choice is paper, the player wins (scissors cut paper). If the computer's choice is rock, the player loses (rock crushes scissors).

  7. Play Again? :

    - After determining and displaying the result of the round (win, lose, or tie), the game asks if you want to play another round.
    - If you choose to play again, the game restarts the loop and you can make a new choice.
    - If you decide not to play again, the game ends and displays a goodbye message.

This project is a good exercise for practicing Python fundamentals like loops, functions, and working with lists and dictionaries.
