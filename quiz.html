Simple text-based Quiz Game

This Python project is a simple text-based quiz game. The game asks multiple-choice questions to the player, evaluates their answers, and then displays the player's score at the end. Below is an explanation of how the different functions work together:

View code

  1. `new_game()` Function :

    Purpose : This is the main function that starts a new quiz game. - Initializes an empty list `guesses` to store the player's answers and `correct_guesses` to count the number of correct answers.
    - Loops through each question in the `questions` dictionary.
    - Displays each question and its corresponding options from the `options` list.
    - Collects the player's guess and stores it in the `guesses` list.
    - Calls the `check_answer()` function to verify if the guess is correct and increments the `correct_guesses` count accordingly.
    - Calls `display_score()` at the end to show the final results.
    - Asks if the player wants to play again using the `play_again()` function.

  2. `check_answer(answer, guess)` Function :

    Purpose : To verify if the player's guess matches the correct answer.
    - Compares the player's guess with the correct answer.
    - Prints "correct" if the guess is right and "Wrong" if it's incorrect.
    - Returns `1` if the answer is correct (to increment the score) or `0` if it's wrong.

  3. `display_score(correct_guesses, guesses)` Function :

    Purpose : To display the final score and a summary of the player's answers.
    - Prints the correct answers and the player's guesses side by side.
    - Calculates the score as a percentage by dividing the number of correct guesses by the total number of questions.
    - Displays the final score.

  4. `play_again()` Function :

    Purpose: To determine if the player wants to restart the game.
    - Prompts the player with "play again: yes/no".
    - If the player types "yes", it calls the `new_game()` function to restart the game.
    - If the player types anything else, it ends the game with a "bye!" message.

  5. Data Structures

    **`questions` Dictionary**: Contains the quiz questions as keys and the correct answer (A, B, C, or D) as values.
    - `options` List : A list of lists, where each sublist contains the multiple-choice options for a corresponding question.

Example of Game Flow:

  • The game starts and asks the first question, displaying the multiple-choice options.
  • The player inputs their guess (A, B, C, or D).
  • The game checks if the guess is correct and continues to the next question.
  • After all questions are answered, the game displays the correct answers, the player's guesses, and their final score.
  • The player is asked if they want to play again.

This project is a good exercise for practicing Python fundamentals like loops, functions, and working with lists and dictionaries.
