Simple Calculator in C

This calculator can perform addition, subtraction, multiplication, division, and modulus operations. Let’s dive into how it works.

View code

  1. Introduction Message :

    - The program starts by displaying a welcome message to the user: *** CALCULATOR ***.

  2. User Input for the First Number :

    - The user is prompted to enter the first number. This input must be an integer.

  3. User Input for the Operator :

    - The user is asked to choose an operator from the following options:
      + for addition
      - for subtraction
      * for multiplication
      / for division
      % for modulus (remainder of division)
    -The program ensures that the operator entered is one of these valid options.

  4. User Input for the Second Number :

    - The user is prompted to enter the second number, which also must be an integer.

  5. Performing the Calculation :

    - Based on the operator entered, the program performs the corresponding arithmetic operation.

  6. Handling Invalid Inputs :

    - If the user enters an invalid operator (anything other than +, -, *, /, %), the program displays an error message indicating that the operator is not valid.
    - The program also includes checks to ensure that division and modulus operations are not attempted with a zero divisor, as this would cause an error.

  7. Ending the Program :

    -After performing the calculation and displaying the result, the program ends. If there were any errors, such as an invalid operator or division by zero, appropriate error messages are displayed before the program terminates.

This project can handle invalid inputs and it supports basic arithmetic operations.This simple calculator in C is a great example of how basic programming concepts can be used to create functional and interactive applications.
